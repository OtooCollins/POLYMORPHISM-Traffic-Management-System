# Smart Traffic Management System

## Description
This project demonstrates polymorphism in Python using inheritance and method overriding. Different traffic devices implement the same `activate()` method but perform different tasks.

## Classes
- TrafficDevice (Parent Class)
- TrafficLight
- SpeedCamera
- PedestrianSignal
- EmergencySiren

## Features
- Inheritance
- Method overriding
- Polymorphism
- Dynamic method dispatch
- Object list iteration

## How to Run
1. Download or clone the project.
2. Open the project in Python.
3. Run the program:

```bash
python traffic_management.py
```

## Sample Output

```text
Traffic Light: Green light ON
Speed Camera: Capturing speeding vehicles
Pedestrian Signal: Walk sign displayed
Emergency Siren: Warning siren activated
```
